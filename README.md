# zoid

**The coding agent that adapts as fast as the ecosystem.**

Terminal-native · Built in Rust · One ~16 MB binary

- **Bring your own tools.** Any MCP server, wired in as first-class tools.
- **Choose your own model.** Local or hosted — your key, your provider.
- **Install a skill set. It becomes a mode.** Extend zoid without waiting on a release.

**→ [strvmarv.github.io/zoid-releases](https://strvmarv.github.io/zoid-releases/)**

## Install

```sh
curl --proto '=https' --tlsv1.2 -LsSf https://github.com/strvmarv/zoid-releases/releases/latest/download/zoid-installer.sh | sh
```

A PowerShell installer and per-platform archives are on the
[latest release](https://github.com/strvmarv/zoid-releases/releases/latest).

Evaluation builds expire 30 days after release — run `zoid update` to stay
current (anonymous, checksum-verified, straight from this repo).

## About this repo

zoid is in beta and its source is private. This repo is the public face: it
hosts the release binaries, the project site, and the plugin catalog.
