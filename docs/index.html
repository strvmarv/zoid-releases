The coding agent that adapts as fast as the ecosystem.

Terminal-native · Built in Rust · One ~16 MB binary

Now in beta · v0.4.0

The right context — not just the recent context. zoid curates the working set, compacts what's verbose, and narrates every move so you can see and undo it.

your tools, your models

Bring your own tools. Choose your own model.

Connect any MCP server's tools alongside zoid's built-ins — configured tools appear automatically. Point zoid at the provider you want: Ollama (local & cloud), Anthropic, or the OpenCode Zen gateway for OpenAI, Google Gemini, and a large model catalog. And on-device semantic recall means nothing has to leave your machine.

extensible by design

Install a skill set. It becomes a mode.

Extend zoid without waiting on a release. :plugin install superpowers installs Superpowers — an open-source skill set by Obra — as a first-class mode: structured TDD, systematic debugging, planning, and code review. Modes are importable skill sets (Chat is always the floor); cycle them with Shift+Tab. The core stays thin; the frontier moves.

Shift+Tab ⇄ CHATSUPERPOWERS
