context economy

The right context — not just the recent context.

zoid continuously curates the model's working set: it drops what stopped mattering, compacts what's verbose, and narrates every move so you can see and undo it. A coding agent lives or dies on what's in the window — zoid makes that a managed resource, measured in tokens.

extensible by design

As the market shifts, so does zoid.

Rich-import any skill set and it becomes a first-class mode — switch between them with Shift+Tab. No hardcoded feature surface, no waiting on a release: drop in a folder (or paste a URL) and zoid gains a capability. The core stays thin; the frontier moves.

the conversation is a database

One conversation, three altitudes.

Because the session is an event log and the UI is a projection over it, you can zoom out to scan the shape of a session — or zoom in to read every token. Same data, rendered at the altitude you need.

no runtime to install

One binary. Cold-starts in milliseconds.
