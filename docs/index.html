your tools, your models

Bring your own tools. Choose your own model.

Connect any MCP server's tools alongside zoid's built-ins — configured tools appear automatically. Point zoid at the provider you want: Ollama (local & cloud) or Anthropic. And on-device semantic recall means nothing has to leave your machine.

extensible by design

As the market shifts, so does zoid.

Rich-import any skill set and it becomes a first-class mode — switch between them with Shift+Tab. One command installs Superpowers, a curated skill set — TDD, systematic debugging, code review, planning — as a ready-to-use mode in a single step. No hardcoded feature surface, no waiting on a release: drop in a folder (or paste a URL) and zoid gains a capability. The core stays thin; the frontier moves.

no runtime to install

One binary. Cold-starts in milliseconds.

A single ~13 MB static binary per platform (Linux x86_64, macOS Apple Silicon, Windows x86_64) — about a 5 MB download — with a ~10 ms cold start and checksum-verified self-update. Nothing to install around it.
